# 🚀 Space Shooter (Python)

A simple text-based Space Shooter game built using basic Python without any external libraries.

## About

This project is a beginner-friendly implementation of a Space Shooter game that runs entirely in the terminal.

The player controls a spaceship and tries to align with the enemy before firing a shot.

## Features

* Move the spaceship left and right
* Shoot at the enemy
* Terminal-based gameplay
* No external libraries required
* Beginner-friendly Python project

## Controls

| Key | Action     |
| --- | ---------- |
| a   | Move Left  |
| d   | Move Right |
| s   | Shoot      |
| q   | Quit Game  |

## Game Symbols

* **A** → Player Spaceship
* **E** → Enemy
* **.** → Empty Space

## How to Run

1. Save the code as `space_shooter.py`
2. Open a terminal in the project folder
3. Run:

```bash
python space_shooter.py
```

## Example

```text
---A--E----

a=left d=right s=shoot q=quit : d

----A-E----

a=left d=right s=shoot q=quit : d

-----AE----

a=left d=right s=shoot q=quit : s

Enemy Destroyed!
```
